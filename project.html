Analog Clock Guessing Game

Java Summer 2023 Education

Built an educational game to help children learn to read analog clocks by guessing the time shown on randomly generated clock faces.

  • Three levels: 1) guess the hour; 2) guess hour and quarter-hour; 3) guess hour and specific minutes.
  • Progression: advance a level after 7 correct guesses in a row; display a congratulations message upon completing level 3.
  • Randomized time generation and input validation tailored to each difficulty level.
